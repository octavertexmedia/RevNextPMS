Quick Start Guide

1. Create Your Account

Sign up for a free 14-day trial. No credit card required.

  1. Visit the registration page
  2. Fill in your hotel information and create your account
  3. Verify your email address
  4. Log in to your dashboard

2. Add Your First Property

Set up your hotel property with room types and rates.

  1. Go to Core Management → Properties in the admin
  2. Click Add Property
  3. Enter property details (name, address, contact info)
  4. Add room types (Standard, Deluxe, Suite, etc.)
  5. Create rate plans for each room type
  6. Set up inventory for the next 365 days

3. Connect OTA Platforms

Integrate with your preferred booking platforms.

  1. Go to Platform Integrations → Integration Platforms
  2. Select the platform you want to connect (e.g., Booking.com)
  3. Click Add Property Integration
  4. Enter your API credentials from the OTA platform
  5. Map your room types and rate plans
  6. Test the connection

4. Configure Synchronization

Set up automatic synchronization of inventory and rates.

  1. Go to Platform Integrations → Property Integrations
  2. Select your integration
  3. Enable Automatic Sync
  4. Set sync frequency (every 5 minutes recommended)
  5. Configure rate update rules
