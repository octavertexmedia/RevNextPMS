Property Management

Complete guide to managing properties, rooms, and rate plans

Property management is the foundation of your channel manager setup. This guide covers everything you need to know about managing properties, room types, and rate plans effectively.

Managing Properties

Adding a New Property

  1. Go to Properties in your dashboard
  2. Click "Add New Property"
  3. Fill in the required information:
    • Basic Information: Name, legal name, property type
    • Address: Complete address with location
    • Contact Details: Phone, email, website
    • Business Information: GSTIN, PAN, timezone, currency
  4. Save the property

Editing Property Details

You can update property information at any time:

  • Navigate to the property you want to edit
  • Click the "Edit" button
  • Update any fields as needed
  • Save your changes

Important: Changes to property details may require re-syncing with connected OTA platforms. The system will notify you if a sync is needed.

Room Types Management

Creating Room Types

Room types define the different categories of rooms at your property:

Room Type Fields

  • Name: Descriptive name (e.g., "Deluxe Sea View Room")
  • Description: Detailed description of the room
  • Max Occupancy: Maximum number of guests
  • Bed Configuration: Number and type of beds
  • Room Size: Square footage or square meters
  • Amenities: WiFi, AC, TV, minibar, etc.
  • Images: Upload room photos

Best Practices for Room Types

  • Create room types that match your OTA listings exactly
  • Use clear, descriptive names that guests will understand
  • Include all relevant amenities to help with search rankings
  • Upload high-quality photos for each room type
  • Keep descriptions accurate and up-to-date

Rate Plans Management

Creating Rate Plans

Rate plans define pricing strategies for your room types:

Rate Plan Configuration

  • Name: Rate plan name (e.g., "Standard Rate", "Non-Refundable")
  • Room Type: Associated room type
  • Base Rate: Default price per night
  • Currency: INR or other supported currencies
  • Cancellation Policy: Free cancellation, non-refundable, etc.
  • Meal Plan: Room only, breakfast, half-board, full-board
  • Length of Stay Discounts: Discounts for longer stays
  • Advance Booking Discounts: Early bird specials

Rate Plan Types

Standard Rate

Your base rate with standard cancellation policy

Non-Refundable

Lower rate with no cancellation option

Early Bird

Discount for advance bookings

Last Minute

Special rates for same-day bookings

Multi-Property Management

If you manage multiple properties, RevNext Channel Manager makes it easy:

  • Centralized Dashboard: View all properties from one place
  • Property-Specific Settings: Each property has its own room types, rates, and integrations
  • Bulk Operations: Apply changes to multiple properties at once
  • Unified Reporting: Generate reports across all properties or for individual properties
  • Role-Based Access: Assign staff to specific properties

Tips and Best Practices

Property Management Best Practices

  • Keep property information accurate and up-to-date
  • Use consistent naming conventions across all properties
  • Regularly review and update room type descriptions
  • Monitor rate plan performance and adjust as needed
  • Set up proper cancellation policies for each rate plan
  • Use high-quality images for all room types
  • Keep inventory updated to avoid overbookings
